# QQmusic_coin_BI 🎵📊

`QQmusic_coin_BI`（乐币看板）是一款专为多账号玩家、粉丝运营及数据站打造的 **QQ 音乐乐币资产管理、花销流水记账与智能兑换规划的 BI 可视化大屏**。

应用采用纯前端架构，数据完全存储于用户本地浏览器（`localStorage`），在保障绝对数据隐私安全的前提下，提供资产存量、充值消耗对比、大额花销预警等多维度的数据可视化分析，并内置了一套实用的**智能折算与充值方案规划器**。

👉 **项目在线演示入口**: [https://qmusicbi.netlify.app/](https://qmusicbi.netlify.app/)

---

## 🚀 核心功能

- 👤 **多账号账本管理 (Accounts)**：
  - 自由创建、编辑或删除多个独立账号（如：日常主号、打榜小号、团队公用数据号）。
  - 每个账号独立记录**当前乐币余额**、**历史总充值**以及自定义备注。
- 💸 **精细化花销记账 (Transactions)**：
  - 记录每笔流水的乐币消耗量，并支持同步记录实际支付的 **RMB 金额**。
  - 动态关联账户，支出时系统自动扣减对应账号的乐币余额。
- 📊 **智能 BI 可视化大屏 (Dashboard)**：
  - **核心 KPI 统计卡片**：实时总览全账号的资产余额、累计总充值、已消耗乐币、平均账号资产等，并带有**高额单次花销预警**。
  - **资产存量分布 (Pie Chart)**：直观展现各个账号的资金占比，迅速锁定主力钱包。
  - **充值与消耗对比 (Bar Chart)**：柱状图直观呈现各账号的资金沉淀与消耗健康度。
  - **大额花销追踪**：多维度筛选并高亮展示高额度、大比重的关键花销历史。
- 🧮 **智能折算与兑换规划器 (Planner)**：
  - 输入目标活动或解锁数字专辑所需的**目标乐币总数**。
  - 系统自动读取全账号当前的乐币总余额，秒级计算出**乐币资金缺口**。
  - 支持根据历史记账数据（充值 RMB 与所得乐币）自动计算**平均充值折扣率**，并给出达成目标最少需要再充值的 **RMB 估算金额**，实现科学精细预算。
- 🔒 **100% 本地隐私安全**：
  - 无后端、无任何第三方审计接口。所有敏感的资产数据和账目流水均通过 `localStorage` 留存在用户本地，绝不上传云端。

## 🛠️ 技术栈

本项目基于最新的一线前端生态开发，保证了极高的渲染性能与丝滑的动态交互：

- **核心框架**: React 19 (`TypeScript`)
- **构建工具**: Vite 6
- **样式框架**: Tailwind CSS v4.0 (基于原生 CSS 变量的极速响应式布局)
- **图表引擎**: Recharts v3.8 (现代化的 SVG 交互式 BI 图表组件库)
- **图标生态**: Lucide React
- **动效反馈**: Motion v12 (多处采用平滑的过渡动画提升数据录入体验)

## 📦 快速开始

请确保你的本地环境已安装了 [Node.js](https://nodejs.org/)（建议版本 v18+）。然后按照以下步骤在本地运行和构建项目：

### 1. 克隆本地仓库
打开终端（Terminal）或命令行工具，运行以下命令将项目克隆到本地：
```bash
git clone [https://github.com/solarshao1006/QQmusic_coin_BI.git](https://github.com/solarshao1006/QQmusic_coin_BI.git)
cd QQmusic_coin_BI
```

### 2. 安装项目依赖
```bash
npm install
```

### 3. 启动本地开发服务器
```bash
npm run dev
```
项目默认将在本地 http://localhost:3000 端口启动，支持 HMR（热更新）。

### 4. 生产环境编译打包
```bash
npm run build
```
编译完成后，会在线上根目录下生成 dist/ 纯静态文件夹，可无缝部署至 Netlify、Vercel、GitHub Pages 或你自己的服务器。

## 💡 使用指南
配置账号：首次使用时，在顶部的“账号管理”模块中，点击 + 添加账号，为你管理的各个乐币账号录入当前的“初始余额”和“历史已充值金额”。

记录流水：当在 QQ 音乐中购买了数字专辑、在直播间送出礼物或者参与打榜消耗乐币时，在“新增花销流水”中建立一笔记录，输入消耗的乐币数量、并可选填本次花费的实际 RMB（系统会自动计算折扣）。

规划预算：若近期有全新的偶像新歌解锁或大规模打榜活动，进入“兑换规划”区域输入目标所需的总乐币。看板会自动根据你全账号的存量余额算出资金缺口，并依据你历史流水的真实充值性价比，反馈给你需要准备的 RMB 预算。
